# Logging Facade 와 Logger
## Logging Facade
    - JCL 
      - Runtime 시에 Logger 를 설정하는 관계로 여러가지 문제점을 발생시킬 여지를 포함하고 있음. 현 시점에서는 잘 사용하지 않음
    - SLF4J
      - Compile 단계에서 Logger 를 지정하여 반영함

## Logger
    - JUL
    - Log4j
    - Log4j2
    - Logback
